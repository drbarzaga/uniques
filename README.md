# Unique Values
Simple NPM package to remove duplicate items in array.

# Installation
```
npm i unique-values
```

```
yarn add unique-values
```

# Usage

```js
var uniqueValues = require("unique-values")

console.log(uniqueValues([1,2,1,1,1,2,3,4,5,3,2,4,5,6]))

// Output -> [1, 2, 3, 4, 5, 6]
```
